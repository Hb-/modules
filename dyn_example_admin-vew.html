Admin List of Dynamic Examples

If you haven't already, please see the user-view page for demonstrations of four approaches to listing items. You may recognize that this admin page uses Approach #2. We could have used any of the other approaches here as well.

Tag in admin-view.xt: 

<xar:data-view object="$mylist" numitems="$items_per_page"
startnum="$startnum" tplmodule="dyn_example"
template="admin_list" sort="$sort"/>

Template invoked by the tag:

modules/dyn_example/xartemplates/objects/showview-dyn_example.xt

Output:

5 Johnny 32 Display | Edit | Delete
6 Nancy 29 Display | Edit | Delete
7 Baby 1 Display | Edit | Delete
8 Your Name Display | Edit | Delete
