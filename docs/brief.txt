1. Designation: Mailer module

2. Purpose: Greater support for Emails in Xaraya

3. Description: 

The Mailer module offers:

- support for text, HTML, BL emails
- choosable headers and footers
- configurations on global or per mail basis
- email alias mails, to support different language versions of a message
- automatic saves of all emails sent

4. Requirments:

- Xaraya 2.2.0 or higher
- Listing property
