Option: convert_fonts_to_styles

If you set this option to true, font size, font family, font color and font background color will be placed on span elements or at the at the selected element, this option is enabled by default since font elements are deprecated and not valid XHTML. When this option is set to true, all font elements will be converted into span elements when the editor initializes. Remember to add "font[face|size|color]" to your extended_valid_elements or valid_elements option.

More information about this subject can be found in this article: What's wrong with FONT FACE

Example of usage of the convert_fonts_to_styles option:

tinyMCE.init({
	...
	convert_fonts_to_styles : false
});
