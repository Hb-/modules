Get a single list with all its items

Parameter: list name

$list = xarModAPIfunc(
    'lists', 'user', 'getlistitems',
    array(
        'list_name'=>$this->listname,
        'listkey'=>'index',
        'itemkey'=>'code',
        'lang_suffix' => $this->lang_suffix
    )
);

Full entry (from header):

 * @param $args['lid'] list ID (optional)

 * @param $args['itemkey'] key for item list array [id]|code (optional)

 * @param $args['items_only'] return just the items, without the list details (default: false)
