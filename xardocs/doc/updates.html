Getting Updates and New Versions

» Patches for the current release of CPAINT can be found on the SourceForge Patch System. (Note: We do not keep patches for previous versions of CPAINT - you will need to download the latest version to get all enhancements.)

» All public releases of CPAINT can be downloaded from SourceForge.

» If you would like to be notified when a new version of CPAINT is released, your best bet is to subscribe to the CPAINT project on Freshmeat. (Note: This will require you to create an account on Freshmeat if you do not already have an account .)
