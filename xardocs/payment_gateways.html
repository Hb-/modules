To set up a new payment gateway, you need to do three things:

  1. Write an API function similar to xaradminapi/authorizenet.php to connect to your payment gateway. You should be able to find sample code in your payment gateway's developer documentation.
  2. Add your payment gateway to the array in xaradminapi/gateways.php.
  3. Add some code in xaradminapi/handlepgresponse.php to handle the response from your payment gateway.

Notes on payment gateways:

Request a free Authorize.net test account: http://developer.authorize.net/testaccount/

PayPal... go here: https://developer.paypal.com/ Click the US or UK link next to Create a Website Payments Pro account. I tested with US.

PaySimple is almost integrated but I have little more work to do on that. They are a reseller of USA ePay so that integration should work for all other resellers.

Of the three, Authorize.net was the easiest to get started with.
